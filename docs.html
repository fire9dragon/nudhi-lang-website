Basic Functions

Below are the basic functions currently supported in Nudhi Lang:

  • nudhi_say "text to print out"
    Prints out the specified text.
    Example: nudhi_say "Hello, World!"
  • nudhi_ask "prompt text"
    Prompts the user to input some text and then prints it.
    Example: nudhi_ask "What's your name?"
  • nudhi_die
    Exits the program.
    Example: nudhi_die
  • nudhi_do "system command"
    Executes a system command.
    Example: nudhi_do "ls" (for listing files in Unix-based systems)
